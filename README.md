## ✨ What is TripKhata?

**TripKhata** (formerly SplitWay) is a premium, modern Progressive Web App (PWA) designed specifically for the Indian travel context. It eliminates the hassle of manually calculating "who-owes-who" during group trips. Track expenses in real-time, get AI-powered insights, and settle up instantly with automatic debt minimization.

---

## 🚀 Key Features

- **⚡ Real-Time Syncing (Firebase)** - Everyone sees updates instantly across all devices. No more "send me the bill screenshot."
- **🤖 TripKhata Buddy (AI)** - Get smart insights on your spending patterns and "Trip Score" using Gemini AI.
- **💰 Smart Settlements** - Our algorithm minimizes the number of transactions required to settle everyone's debt.
- **📱 Install as App** - Full PWA support. Add to home screen on iOS & Android for a native app feel.
- **📊 Aesthetic & Dark-Themed** - A premium, modern UI using **Plus Jakarta Sans** and high-end glassmorphism.
- **📄 PDF Export** - Generate beautiful trip reports to share in WhatsApp groups.
- **✈️ Offline Ready** - Works perfectly on mountain treks or remote beaches with zero network; syncs as soon as you're back.
- **🔒 Privacy First** - No phone numbers, no emails, no personal data. Just your name and the fun.

## 📲 How to Use

### Quick Start
1. **Visit** [tripkhata.netlify.app](https://tripkhata.netlify.app)
2. **Create Trip**: Name your adventure and set a base currency.
3. **Invite Friends**: Share the unique 6-digit Room ID.
4. **Add Expenses**: Log bills as they happen.
5. **Settle Credits**: One-tap see who owes what.

### Install as Mobile App

- **iOS**: Safari → Share → Add to Home Screen  
- **Android**: Chrome → Menu (⋮) → Add to Home Screen

## 🛠️ Tech Stack

- **Framework**: React 18 + Vite
- **Language**: TypeScript
- **Styling**: Tailwind CSS + Vanilla CSS (Custom Glassmorphism)
- **Backend**: Firebase Firestore (Mumbai Region for ultra-low latency)
- **Animations**: Framer Motion
- **AI**: Google Gemini Pro (via Groq API)
- **PWA**: Vite PWA Plugin + Service Workers

---

## 🔐 Security & Architecture

- **Firebase Firestore**: Secure real-time database with strict security rules.
- **Local-First**: Robust offline queue for a seamless experience in low-connectivity zones.
- **Environment Driven**: No hardcoded API keys or secrets; full CI/CD via Netlify.
- **Zero Tracker Policy**: No third-party ads or user tracking.
